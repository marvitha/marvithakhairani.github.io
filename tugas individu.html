    		W3Schools merupakan situs pembelajaran web development yang meneyediakan banyak tutorial dan referensi pengembangan web untuk berbagai teknologi web. Tidak hanya itu saja layanan yang diberikan oleh web ini ada berbagai layanan menarik yang bisa di gunakan untuk belajar Pemograman. Ada banyak sekali tutorial pembelajaran pengembangan web mulai dari HTML, web building, data analiytic,sampai ke backend.
    Berikut beberapa bahasa pemograman yang bisa kamu pelajari dalam W3Schools:

    1.CSS
    CSS (Cascading Style Sheet) merupakan bahasa pemrograman yang dikembangkan oleh W3C (World Wide Web Consurtium) yang pada umumnya digunakan untuk mengontrol atau mengatur tampilan dari sebuah web atau blog. CSS adalah bahasa pemrograman yang wajib digunakanuntuk membangun sebuah website agar tampak lebih rapi dan enak dilihat. Biasanya skrip CSS diletakan bersama-sama dengan dokumen HTML ataupun dokumen tersendiri.

    2.C
    Bahasa C merupakan salah satu yang populer yang umum digunakan untuk membuat website maupun aplikasi desktop. Struktur bahasa C memiliki kemiripan dengan C++ dan C#, karena memang kedua bahasa pemrograman tersebut dikembangkan oleh bahasa C sendiri, bahkan banyak bahasa pemrograman lainnya yang syntax-nya meniru bahasa ini.

    3.C++
    Bahasa pemrograman yang satu ini masih satu keluarga dengan bahasa C, artinya bahasa C++ banyak mengadopsi sintax-sintax dari bahasa C, walaupun mirip tapi tidak sama. C++ adalah bahasa pemrograman terbaik pilihan para web programmer maupun desktop programmer dan sudah banyak website-website terkenal yang dibangun dengan bahasa ini.

    4.JavaScript
    Javascript merupakan bahasa pemrograman yang terbaik dan populer untuk membangun sebuah web. Javascript (atau biasa disingkat dengan JS) sendiri biasa digunakan untuk membuat website menjadi tampak lebih dinamis dan interaktif. Bahasa pemrogaman ini merupakan salah satu bahasa pemrograman yang paling banyak dipakai para web developer karena mudah dibuat dan tidak memerlukan compiler.

    W3Schools sudah banyak menyediakan materi yang kaya dan terstruktur mulai dari menggunakan model kode yang jelas, yang membuat platform online ini cocok bagi pemula yang ingin belajar mulai dari dasar pengembangan web.
